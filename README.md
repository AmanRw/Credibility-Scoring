# Credibility-Scoring
Browser Extension for "Credibility Scoring": A plugin that analyzes news articles in real-time, providing an "Integrity Score" based on source history, author credibility, and linguistic cues like "clickbait" headlines or emotional triggers.
# Link
The original project is available at https://www.figma.com/design/65IVepxf71tEdmRjnPqK2B/Credibility-Scoring-Extension.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
